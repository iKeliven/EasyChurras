# EasyChurras


# Ideia

O EasyChurras é uma plataforma mobile que conecta clientes que desejam fazer churrascos a churrasqueiros profissionais, proporcionando tudo o que o cliente precisa para ter um churrasco.

## Para quem a plataforma é destinada?

A plataforma se destina à profissionais churrasqueiros que estão em busca de novas oportunidades no mercado e á pessoas que queiram fazer churrascos, mas não sabem ou não querem ter a necessidade de comprar os insumos, organizar e preparar a refeição.
